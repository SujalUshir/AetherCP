# AI Workspace

This folder stores collaboration history with LLMs.

## Save Here

- Prompts that produced useful work.
- Generated outputs worth preserving.
- Summaries of reasoning.
- Implementation notes after manual edits.
- Bug fixing sessions.
- Feature specifications.

## Do Not Save

- API keys.
- Personal credentials.
- Large unrelated transcripts.
- Generated code that was rejected and has no learning value.

## Recommended Flow

1. Save the prompt in `ai/prompts/`.
2. Save the useful output in `ai/outputs/`.
3. Implement manually or with Codex.
4. Save final notes in `ai/implementation-notes/`.
5. Update `docs/TASKS.md` and `docs/CHANGELOG.md`.
